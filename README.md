# Join – Task Management Application

## Overview

Join is a modern task management application inspired by the Kanban methodology. It enables users to create, organize, and manage tasks efficiently through an intuitive drag-and-drop interface. The application supports task categorization, user assignment, priority management, and responsive design for seamless usage across different devices.

## Features

* Create, edit, and delete tasks
* Drag-and-drop task management
* Kanban board workflow
* User assignment and contact management
* Task prioritization
* Category management
* Responsive design for desktop and mobile devices
* User-friendly interface
* Firebase backend integration

## Installation

### Prerequisites

* Node.js (latest LTS version recommended)
* Angular CLI

### Clone Repository

```bash
git clone https://github.com/thorbenschreyer/join.git
```

### Navigate to Project Directory

```bash
cd join
```

### Install Dependencies

```bash
npm install
```
